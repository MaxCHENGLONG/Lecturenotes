About Reinforcement Learning Definition

Agent

  • Agent observes environments state and takes an action →\rightarrow→﻿
  • Agent receives a reward and environments state changes →\rightarrow→﻿
  • Agent observes new state and takes an action.

State, StS_tSt​﻿

  • State, StS_tSt​﻿: A representation of the environment at time ttt﻿.
  • State space, S\mathcal{S}S﻿: The set of all possible states.
  • StS_tSt​﻿ will depend on what happended in the past (before time ttt﻿).
  • StS_tSt​﻿ contains all information, besides future actions AtA_tAt​﻿, that is relevant for predicting the future at time t.t.t.﻿

Reward, RtR_tRt​﻿

  • Reward, RtR_tRt​﻿: A scalar signal that tells it how it is doing at step t.t.t.﻿
  • Goal : Maximize the long-time cumulative reward.

Action, AtA_tAt​﻿

  • Action, AtA_tAt​﻿ : The agent can affect the state by actions.
  • Action space, A\mathcal{A}A﻿ : The set of all possible actions.
  • Without feedback : Given S0S_0S0​﻿ we can pre-compute actions A0,A1,⋯A_0,A_1,\cdotsA0​,A1​,⋯﻿
  • Feedback: Decide on action AtA_tAt​﻿ after we have observed St.S_t.St​.﻿
  • Instead of trying to find good actions, we will try to find a good policy.

  • Learn a model from experience.
  • Use model to find good policy and/or predictions.

Model-free RL:

  • Learn policy and/or predictions directly, without first learning a model.
