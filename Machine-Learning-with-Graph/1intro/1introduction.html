Definition Graphs are a general language for describing and analyzing entities with relations/interactions.

二分图 Bipartite Graph

  • Bipartite graph is a graph whose nodes can be divided into two disjoint sets UUU﻿ and VVV﻿ such that every link connects a node in UUU﻿ to one in VVV﻿; that is, UUU﻿ and VVV﻿ are independent sets.

    • Authors-to-Papers (they authored)

    • Users-to-Movies (they rated)

  • “Folded” networks
    • Author collaboration networks
    • Movie co-rating networks

Link-Level Prediction Task

Link-level prediction task example

  • The task is to predict new/missing/unknown links based on the existing links.
  • At test time, node pairs (with no existing links) are ranked, and top KKK﻿ node pairs are predicted.
  • Task: Make a prediction for a pair of nodes.

图的边预测 (Link Prediction as a Task)

Two formulations of the link prediction task:

  1. Links missing at random:

    Remove a random set of links and then aim to predict them

  1. Links over time:

    Given G[t0,t0′]G[t_0,t'_0]G[t0​,t0′​]﻿ a graph defined by edges up to time t0′t'_0t0′​﻿, output a ranked list LLL﻿ of edges (not in G[t0,t0′]G[t_0,t'_0]G[t0​,t0′​]﻿) that are predicted to appear in time G[t1,t1′]G[t_1,t'_1]G[t1​,t1′​]﻿
